
# IMA Market Project

A complete e-commerce website for buying products across Clothing, Electronics, Home & Living, and Sports categories.

## Features

- User Login & Signup
- Product categories: Clothing, Electronics, Home & Living, Sports
- Add to cart functionality
- Generate bill / invoice
- Contact form with email
- Newsletter subscription
- Responsive design using Tailwind CSS

## Technologies Used

- PHP (Backend)
- MySQL (Database)
- HTML, CSS, Tailwind CSS
- JavaScript
